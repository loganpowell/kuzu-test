# Node.js Benchmarks

## Setup

```bash
cd benchmarks/nodejs
npm install
```

## Run Benchmarks

### Create Schema

```bash
npm run schema
```

### Measure Library Size

```bash
npm run measure-size
```

### Benchmark Loading Performance

```bash
npm run benchmark-loading
```

### Benchmark Query Performance

```bash
npm run benchmark-queries
```

## Requirements

- Node.js 18+ recommended
- KuzuDB Node.js bindings will be installed via npm
